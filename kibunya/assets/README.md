# assets/

Expo が必要とするアセット画像をここに配置します。

必要なファイル:
- `icon.png` — アプリアイコン (1024x1024, PNG)
- `splash.png` — スプラッシュ画面 (1284x2778 推奨, PNG)
- `adaptive-icon.png` — Android適応アイコンの前景 (1024x1024, PNG, 透過背景)
- `favicon.png` — Web用 (48x48, 任意)

推奨ツール:
- https://www.figma.com/ でデザイン
- https://easyappicon.com/ でリサイズ
- https://icon.kitchen/ でadaptive-icon生成
